# close项目

类型<br>
信息编辑与图片存储


	
简介：
项目致力于帮助人们更好的书写生活
‘更好的书写’以圈出框架为开始：人们建立一个又一个框架 然后在框架中书写生活
‘框架’ 的建立 使得信息的书写更具创造性

‘信息的形式’只有三种 文字 图片 图册
其中‘图册’包括五种：图片 音频 动图 视频 链接
通过建立‘图册’以分类图片 通过分离大量图片与文章的形式 以更好的书写生活
